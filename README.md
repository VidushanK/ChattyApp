# ChattyApp
Instant Messaging app harnassing React and WebSockets Edit

## Installation:
```
npm install
npm start
open http://localhost:3000
```
## Dependencies:

* React
* Webpack
* Node-uuid
* [babel-loader](https://github.com/babel/babel-loader)
* [webpack-dev-server](https://github.com/webpack/webpack-dev-server)
